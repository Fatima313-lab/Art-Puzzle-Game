# Doraemon's Art Atelier

A Vite + React  puzzle game where players drag  art gadgets into matching shadow slots before the clock runs out.

## Features

- 3D interactive game using `@react-three/fiber` and `@react-three/drei`
- Difficulty modes with improved UI icons and scoring
- Animated combo score effects and level progression
- Bootstrap Icons powered controls and HUD

## Requirements

- Node.js 
- npm

## Installation

From the `art-puzzle-game/` folder:

```bash
npm install
```

## Run the app

```bash
npm run dev
```

Open the local Vite URL shown in the terminal, usually `http://localhost:5173`.

## Build for production

```bash
npm run build
```

## Scripts

- `npm run dev`  start the Vite dev server
- `npm run build`  build the app for production
- `npm run preview`  preview the built app locally
- `npm run lint`  run ESLint checks

## Gameplay

- Drag each floating art piece into its matching shadow outline
- Complete all pieces before time expires
- Achieve higher score with faster matches and combos
- Advance through harder levels as you win

## Dependencies

- `react`, `react-dom`
- `@react-three/fiber`
- `@react-three/drei`
- `three`
- `bootstrap-icons`
